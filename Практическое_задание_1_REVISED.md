# ПРАКТИЧЕСКОЕ ЗАНЯТИЕ 1

## РАЗРАБОТКА ДОРОЖНО-КЛИМАТИЧЕСКОГО ГРАФИКА ПРИ СТРОИТЕЛЬСТВЕ АВТОМОБИЛЬНОЙ ДОРОГИ

### 1.1. Общие положения

Построение дорожно-климатического графика необходимо для определения сроков производства дорожно-строительных работ с учётом территориального расположения объекта строительства автомобильной дороги в интервалах между весенней и осенней распутицами.

Все дорожно-строительные работы разделены на группы в зависимости от температуры воздуха, благоприятной для проведения данного вида работ. Выполнение работ при других температурных условиях приводит к дополнительным затратам или вынужденным простоям.

### 1.2. Источники климатических данных

При выполнении учебной работы климатические данные для построения дорожно-климатического графика следует принимать из следующих источников:

1. **СП 131.13330.2020 «Строительная климатология»** (актуализированная редакция СНиП 23-01-99*):
   - Таблица 5 — средняя месячная и годовая температура воздуха для населённых пунктов РФ;
   - Таблица 6 — даты наступления средних суточных температур выше и ниже определённых пределов и число дней с температурой, превышающей эти пределы;
   - Раздел 10 (Таблица 8) — среднее месячное и годовое количество осадков.

2. **Справочники по климату СССР** (Гидрометеоиздат):
   - Часть 2 «Температура воздуха и почвы» — таблицы средних месячных температур, дат перехода среднесуточной температуры через заданные пределы, глубины промерзания почвы;
   - Часть 4 «Влажность воздуха, атмосферные осадки и снежный покров» — число дней с осадками более 5 мм/сут, средняя декадная высота снежного покрова, число дней с метелями.

3. **СП 20.13330.2016 «Нагрузки и воздействия»** — данные о снеговых нагрузках (при необходимости).

> **Примечание.** В реальном проектировании основой для построения дорожно-климатического графика служат результаты инженерно-гидрометеорологических изысканий, выполняемых в соответствии с СП 47.13330.2016 «Инженерные изыскания для строительства. Основные положения». Данные изысканий включают наблюдения метеостанций за многолетний период (не менее 30 лет), что обеспечивает статистическую достоверность климатических параметров для конкретного района строительства. Для учебных целей допускается использование обобщённых данных из перечисленных выше нормативных источников.

### 1.3. Методика определения основных климатических параметров

#### 1.3.1. Средняя месячная температура воздуха

Средняя месячная температура воздуха определяется по таблице 5 СП 131.13330.2020 или по таблице 1 части 2 Справочника по климату СССР для ближайшей к району строительства метеостанции. Данные заносятся в табл. 1.4.

#### 1.3.2. Даты перехода среднесуточной температуры через заданные пределы

Даты наступления среднесуточных температур выше и ниже определённых пределов (0, +5, +10, +15 °С) определяются по таблице 6 СП 131.13330.2020 или по таблице 5 части 2 Справочника по климату СССР. Данные заносятся в табл. 1.8.

#### 1.3.3. Глубина промерзания грунта

Среднемаксимальная (средняя из максимальных за зиму) глубина промерзания грунта определяется по таблице 8 части 2 Справочника по климату СССР (столбец «среднее максимальное» — Hпр). Данные по месяцам заносятся в табл. 1.7.

**Порядок определения:**
1. Найти в Справочнике по климату СССР (часть 2) населённый пункт, соответствующий варианту задания.
2. В таблице 8 «Глубина промерзания почвы» найти строку с данными по месяцам (октябрь — май).
3. Значение в столбце «Средняя из наибольших» принять за расчётную глубину промерзания hпр (см).

При отсутствии данных для конкретного пункта допускается определять нормативную глубину промерзания расчётным путём по СП 22.13330.2016 «Основания зданий и сооружений» (п. 5.5.3) по формуле:

d_fn = d_0 · √Mt,

где d_0 — величина, зависящая от типа грунта (для суглинков и глин — 0,23 м; для супесей, мелких и пылеватых песков — 0,28 м; для песков средней крупности и крупных — 0,30 м; для крупнообломочных — 0,34 м); Mt — безразмерный коэффициент, равный сумме абсолютных значений среднемесячных отрицательных температур за зимний период.

#### 1.3.4. Число дней с осадками более 5 мм/сут

Число дней с осадками интенсивностью более 5 мм/сут определяется по таблице 8 раздела «Осадки» части 4 Справочника по климату СССР. Данные помесячно заносятся в табл. 1.5.

**Порядок определения:**
1. В части 4 Справочника по климату СССР найти соответствующий населённый пункт.
2. В разделе «Осадки», таблица 8 «Число дней с осадками различной величины» выбрать строку «≥ 5,0 мм».
3. Значения по месяцам и за год перенести в табл. 1.5.

#### 1.3.5. Число дней с метелями

Число дней с метелями определяется по таблице раздела «Метели» части 3 Справочника по климату СССР «Ветер». Данные используются как дополнительный показатель неблагоприятных погодных условий при оценке строительного сезона для нулевой группы работ в зимний период. Дни с метелями учитываются при определении нерабочих дней по климатическим условиям для работ, выполняемых в зимнее время (расчистка полосы отвода, сосредоточенные земляные работы).

#### 1.3.6. Высота снежного покрова

Средняя декадная высота снежного покрова определяется по таблице 1 раздела «Снежный покров» части 4 Справочника по климату СССР. Данные заносятся в табл. 1.6.

### 1.5. Определение сроков распутицы

К линейным земляным работам приступают после оттаивания грунта и окончания весенней распутицы, а заканчивают до начала осенней распутицы. Для связных грунтов даты начала весенней распутицы (z_н^вр) и окончания (z_к^вр) определяются по следующим формулам:

**z_н^вр = z_0 + 5/α** (1.1)

**z_к^вр = z_н^вр + 0,7·h_пр / α** (1.2)

где:
- z_0 — дата перехода температуры воздуха весной через 0 °С, принимаемая по данным справочника по климату или СП 131.13330.2020 (таблица 6);
- α — климатический коэффициент, характеризующий скорость оттаивания грунта, см/сут (табл. 1.2);
- h_пр — среднемаксимальная (средняя из максимальных за зиму) глубина промерзания грунта в данном районе, принимаемая по справочнику по климату (часть 2, таблица 8), см.

Дата начала осенней распутицы (z_н^ор) может быть приурочена к среднемесячной температуре воздуха от +3 до +5 °С, а дата окончания осенней распутицы (z_к^ор) — к переходу среднесуточной температуры через 0 °С в осенний период.

Продолжительность в днях весенней (Т_вр) и осенней (Т_ор) распутицы определяется по формулам:

**Т_вр = z_к^вр − z_н^вр** (1.3)

**Т_ор = z_к^ор − z_н^ор** (1.4)

### 1.6. Состав дорожно-климатического графика

Для определения календарной продолжительности дорожно-строительных работ на основании данных о погодно-климатических условиях строят дорожно-климатический график, на котором должны быть отображены следующие данные:

1. Среднемесячная температура воздуха (кривая за 12 месяцев).
2. Глубина промерзания грунта (кривая нарастания и убывания за зимний период).
3. Высота снежного покрова (кривая за зимний период по декадам).
4. Число дней с осадками более 5 мм (столбчатая диаграмма по месяцам).
5. Продолжительность светового дня (кривая за 12 месяцев).
6. Даты весенней и осенней распутицы (отмечаются вертикальными линиями или заштрихованными зонами).
7. Температурные границы для групп работ (горизонтальные линии на уровнях 0, +5, +10, +15 °С).

**Блок 4 (построение графика).** Прямоугольник с описанием структуры чертежа:
- Ось X — месяцы года (январь–декабрь), разделённые на декады.
- Верхняя зона: кривая температуры (левая ось Y — °С) с горизонтальными линиями температурных пределов (0, +5, +10, +15 °С).
- Средняя зона: столбчатая диаграмма осадков (дни с осадками > 5 мм/сут) и кривая продолжительности светового дня.
- Нижняя зона: кривые глубины промерзания (вниз от оси) и высоты снежного покрова (вверх от оси).

**Блок 5 (определение строительного сезона).** Таблица-результат: для каждой группы работ (0, I, II, III, IV) указываются дата начала, дата окончания и продолжительность строительного сезона.

**Рекомендации по выполнению в AutoCAD:**
- Формат листа: А3 (горизонтальная ориентация) или А2.
- Масштаб оси времени: 1 месяц = 40 мм (итого 480 мм на 12 месяцев).
- Верхняя зона (температура): высота 80 мм, масштаб 1 °С = 2 мм (диапазон от −25 до +25 °С).
- Средняя зона (осадки): высота 40 мм.
- Нижняя зона (промерзание и снег): высота 60 мм, масштаб глубины промерзания 1 см = 0,3 мм.
- Температурные линии (0, +5, +10, +15 °С) выполнять штриховыми линиями разного цвета.
- Период распутицы — штриховка под углом 45° (ANSI31).
- Рамка и штамп — по ГОСТ 2.104.

### 1.11. Алгоритм расчёта периода распутицы

Ниже приведён пошаговый алгоритм расчёта продолжительности весенней и осенней распутицы.

#### Шаг 1. Определение исходных данных

- Определить по табл. 1.8 дату перехода среднесуточной температуры воздуха весной через 0 °С → **z_0**.
- Определить по табл. 1.2 климатический коэффициент α (см/сут) для района строительства.
- Определить по табл. 1.7 среднемаксимальную глубину промерзания грунта → **h_пр** (см).

#### Шаг 2. Расчёт дат весенней распутицы

- Дата начала весенней распутицы:
  **z_н^вр = z_0 + 5/α** (дни после z_0).
- Дата окончания весенней распутицы:
  **z_к^вр = z_н^вр + 0,7 · h_пр / α** (дни после z_н^вр).
- Продолжительность весенней распутицы:
  **Т_вр = z_к^вр − z_н^вр = 0,7 · h_пр / α** (дни).

#### Шаг 3. Определение дат осенней распутицы

- Дата начала осенней распутицы **z_н^ор**: определяется как дата перехода среднесуточной температуры в осенний период через +3…+5 °С (по табл. 1.8 — интерполяцией между датами перехода через +5 °С и 0 °С).
- Дата окончания осенней распутицы **z_к^ор**: дата перехода среднесуточной температуры через 0 °С осенью (по табл. 1.8).
- Продолжительность осенней распутицы:
  **Т_ор = z_к^ор − z_н^ор** (дни).

#### Шаг 4. Пример расчёта

*Район строительства: Тамбовская область.*

Исходные данные:
- z_0 = 29 марта (переход через 0 °С весной, по Справочнику по климату);
- α = 3,5 см/сут (табл. 1.2);
- h_пр = 95 см (по Справочнику по климату, часть 2).

Расчёт:
1. z_н^вр = 29 марта + 5/3,5 = 29 марта + 1,4 ≈ 30 марта.
2. z_к^вр = 30 марта + 0,7 · 95 / 3,5 = 30 марта + 19 = 18 апреля.
3. Т_вр = 18 апреля − 30 марта = 19 дней.

Осенняя распутица:
- z_н^ор = 18 октября (дата перехода через +5 °С осенью);
- z_к^ор = 3 ноября (дата перехода через 0 °С осенью);
- Т_ор = 3 ноября − 18 октября = 16 дней.

#### Шаг 5. Проверка результатов

Рекомендуется проверить, что:
- Продолжительность весенней распутицы для средней полосы России составляет 15–30 дней.
- Продолжительность осенней распутицы составляет 10–25 дней.
- Суммарный строительный сезон для I группы работ составляет 150–210 дней в зависимости от района строительства.
