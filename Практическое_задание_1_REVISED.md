# ПРАКТИЧЕСКОЕ ЗАНЯТИЕ 1

## РАЗРАБОТКА ДОРОЖНО-КЛИМАТИЧЕСКОГО ГРАФИКА ПРИ СТРОИТЕЛЬСТВЕ АВТОМОБИЛЬНОЙ ДОРОГИ

### 1.1. Общие положения

Построение дорожно-климатического графика необходимо для определения сроков производства дорожно-строительных работ с учётом территориального расположения объекта строительства автомобильной дороги в интервалах между весенней и осенней распутицами.

Все дорожно-строительные работы разделены на группы в зависимости от температуры воздуха, благоприятной для проведения данного вида работ. Выполнение работ при других температурных условиях приводит к дополнительным затратам или вынужденным простоям. Классификация работ с учётом температуры воздуха приведена в табл. 1.1.

Дорожно-климатический график является базовым документом для разработки линейного календарного графика производства работ. На его основе определяются границы строительного сезона по каждой группе работ, что позволяет установить последовательность и продолжительность выполнения технологических процессов при составлении проекта организации строительства (ПОС) и проекта производства работ (ППР).

Для учебных целей допускается использование обобщённых данных из перечисленных выше нормативных источников.

### 1.3. Методика определения основных климатических параметров

Средняя месячная температура определяется по таблице 5 СП 131.13330.2020 или по таблице 1 части 2 Справочника по климату СССР для ближайшей к району строительства метеостанции. Данные заносятся в табл. 1.4.

#### 1.3.2. Даты перехода среднесуточной температуры через заданные пределы

Даты наступления среднесуточных температур выше и ниже определённых пределов (0, +5, +10, +15 °С) определяются по таблице 6 СП 131.13330.2020 или по таблице 5 части 2 Справочника по климату СССР. Данные заносятся в табл. 1.8.

#### 1.3.3. Глубина промерзания грунта

Глубина промерзания является критическим фактором для определения даты начала линейных земляных работ: разработка грунта возможна только после его полного оттаивания на глубину, достаточную для работы землеройной техники.

Среднемаксимальная (средняя из максимальных за зиму) глубина промерзания грунта (h_пр) определяется по таблице 8 части 2 Справочника по климату СССР (столбец «среднее из наибольших»). Данные по месяцам заносятся в табл. 1.7.

**Порядок определения:**
1. В Справочнике по климату СССР (часть 2) найти населённый пункт, соответствующий варианту задания (район строительства из табл. 2.1).
2. В таблице 8 «Глубина промерзания почвы» найти строку с данными по месяцам (октябрь — май).
3. Значение в столбце «Средняя из наибольших» принять за расчётную глубину промерзания h_пр (см).

При отсутствии данных для конкретного пункта допускается определять нормативную глубину промерзания расчётным путём по СП 22.13330.2016 (п. 5.5.3):

**d_fn = d_0 · √M_t,**

где d_0 — величина, зависящая от типа грунта (для суглинков и глин — 0,23 м; для супесей, мелких и пылеватых песков — 0,28 м; для песков средней крупности и крупных — 0,30 м; для крупнообломочных — 0,34 м); M_t — безразмерный коэффициент, равный сумме абсолютных значений среднемесячных отрицательных температур за зимний период.

#### 1.3.4. Число дней с осадками более 5 мм/сут

Дни с осадками интенсивностью ≥ 5 мм/сут являются нерабочими по климатическим условиям и учитываются при расчёте технологических перерывов и определении директивного срока строительства.

Данные определяются по таблице 8 раздела «Осадки» части 4 Справочника по климату СССР. Значения помесячно заносятся в табл. 1.5.

**Порядок определения:**
1. В части 4 Справочника по климату найти соответствующий населённый пункт.
2. В разделе «Осадки», таблица 8 «Число дней с осадками различной величины», выбрать строку «≥ 5,0 мм».
3. Значения по месяцам и за год перенести в табл. 1.5.

#### 1.3.5. Число дней с метелями

Число дней с метелями определяется по таблице раздела «Метели» части 3 Справочника по климату СССР «Ветер». Данные используются как дополнительный показатель неблагоприятных погодных условий для обоснования технологических перерывов при оценке строительного сезона нулевой группы работ в зимний период.

Дни с метелями учитываются при определении нерабочих дней по климатическим условиям для работ, выполняемых в зимнее время (расчистка полосы отвода, сосредоточенные земляные работы).

#### 1.3.6. Высота снежного покрова

Средняя декадная высота снежного покрова определяется по таблице 1 раздела «Снежный покров» части 4 Справочника по климату СССР. Данные заносятся в табл. 1.6.

### 1.5. Определение сроков распутицы

К линейным земляным работам приступают после оттаивания грунта и окончания весенней распутицы, а заканчивают до начала осенней распутицы. Для связных грунтов даты начала весенней распутицы (z_н^вр) и окончания (z_к^вр) определяются по следующим формулам:

**z_н^вр = z₀ + 5/α,** (1.1)

**z_к^вр = z_н^вр + 0,7·h_пр / α,** (1.2)

где z₀ — дата перехода температуры воздуха весной через 0 °С, принимаемая по данным справочника по климату или СП 131.13330.2020 (табл. 6); α — климатический коэффициент, характеризующий скорость оттаивания грунта, см/сут (табл. 1.2); h_пр — среднемаксимальная (средняя из максимальных за зиму) глубина промерзания грунта в данном районе, принимаемая по справочнику по климату (часть 2, табл. 8), см.

Дата начала осенней распутицы (z_н^ор) может быть приурочена к среднемесячной температуре воздуха от +3 до +5 °С, а дата окончания осенней распутицы (z_к^ор) — к переходу среднесуточной температуры через 0 °С в осенний период.

Продолжительность в днях весенней (Т_вр) и осенней (Т_ор) распутицы определяется по формулам:

**Т_вр = z_к^вр − z_н^вр,** (1.3)

**Т_ор = z_к^ор − z_н^ор.** (1.4)

### 1.6. Состав и структура дорожно-климатического графика

Для определения календарной продолжительности дорожно-строительных работ на основании данных о погодно-климатических условиях строят дорожно-климатический график, на котором должны быть отображены:

1. Среднемесячная температура воздуха (кривая за 12 месяцев).
2. Глубина промерзания грунта (кривая нарастания и убывания за зимний период).
3. Высота снежного покрова (кривая за зимний период по декадам).
4. Число дней с осадками более 5 мм/сут (столбчатая диаграмма по месяцам).
5. Продолжительность светового дня (кривая за 12 месяцев).
6. Даты весенней и осенней распутицы (вертикальные линии или заштрихованные зоны).
7. Температурные границы для групп работ (горизонтальные линии на уровнях 0, +5, +10, +15 °С).

С помощью дорожно-климатического графика и на основании сопоставления погодно-климатических данных определяют продолжительность строительного сезона по каждой группе работ. Полученные сроки используются далее при составлении линейного календарного графика производства работ, определении директивных сроков строительства и расчёте потребности в ресурсах.

### 1.10. Алгоритм расчёта периода распутицы

Ниже приведён пошаговый алгоритм расчёта продолжительности весенней и осенней распутицы.

**Шаг 1. Определение исходных данных.**
- По табл. 1.8 определить дату перехода среднесуточной температуры воздуха весной через 0 °С → **z₀**.
- По табл. 1.2 определить климатический коэффициент **α** (см/сут) для района строительства.
- По табл. 1.7 определить среднемаксимальную глубину промерзания грунта → **h_пр** (см).

**Шаг 2. Расчёт дат весенней распутицы.**
- Дата начала: **z_н^вр = z₀ + 5/α** (дни после z₀).
- Дата окончания: **z_к^вр = z_н^вр + 0,7·h_пр / α**.
- Продолжительность: **Т_вр = z_к^вр − z_н^вр = 0,7·h_пр / α** (дни).

**Шаг 3. Определение дат осенней распутицы.**
- Дата начала осенней распутицы **z_н^ор**: дата перехода среднесуточной температуры в осенний период через +3…+5 °С (по табл. 1.8 — интерполяцией между датами перехода через +5 °С и 0 °С).
- Дата окончания осенней распутицы **z_к^ор**: дата перехода среднесуточной температуры через 0 °С осенью.
- Продолжительность: **Т_ор = z_к^ор − z_н^ор** (дни).

**Шаг 4. Пример расчёта (Тамбовская область, г. Тамбов).**

Исходные данные:
- z₀ = 29 марта (переход через 0 °С весной);
- α = 3,5 см/сут (табл. 1.2);
- h_пр = 95 см (Справочник по климату, часть 2).

Расчёт весенней распутицы:
1. z_н^вр = 29 марта + 5/3,5 ≈ 29 марта + 1 день ≈ 30 марта.
2. z_к^вр = 30 марта + 0,7 × 95 / 3,5 = 30 марта + 19 дней = 18 апреля.
3. Т_вр = 19 дней.

Осенняя распутица:
- z_н^ор = 18 октября (переход через +5 °С осенью);
- z_к^ор = 3 ноября (переход через 0 °С осенью);
- Т_ор = 16 дней.

**Шаг 5. Контроль результатов.**
- Продолжительность весенней распутицы для средней полосы России — 15–30 дней.
- Продолжительность осенней распутицы — 10–25 дней.
- Суммарный строительный сезон для I группы работ — 150–210 дней.

# ПРАКТИЧЕСКОЕ ЗАНЯТИЕ 2

## РАСЧЁТ ШИРИНЫ ПОЛОСЫ ОТВОДА ЗЕМЛИ ПОД СТРОИТЕЛЬСТВО АВТОМОБИЛЬНОЙ ДОРОГИ И ОПРЕДЕЛЕНИЕ ОБЪЁМОВ ГРУНТА ПРИ ВОЗВЕДЕНИИ ЗЕМЛЯНОГО ПОЛОТНА С РАЗРАБОТКОЙ ГРУНТА ИЗ БОКОВЫХ РЕЗЕРВОВ

### 2.1. Общие положения

Ширина полосы отвода под строительство автомобильной дороги устанавливается в зависимости от категории дороги, количества полос движения, высоты насыпи (глубины выемки) и наличия боковых резервов.

При высоте насыпи не более 2 м ширина полосы отвода определяется от края полосы проезжей части дороги с учётом расстояния боковой видимости:

**В_п.отв = В_пол + 2В_вид, м,** (2.1)

где В_пол — ширина полосы движения, м; В_вид — ширина полосы видимости, м (для дорог I–III категорий В_вид = 25 м, для дорог IV категории — 15 м).

При наличии кюветов ширина полосы отвода определяется с учётом ширины кювета по верхней части и прилегающей полосы:

**В_п.отв = В_н + 2В_к + 2П_п, м,** (2.2)

где В_н — ширина насыпи понизу, м; В_к — ширина кювета поверху, м; П_п — ширина прилегающей полосы, принимается 3,0 м.

При определении полосы отвода для выемки учитывается ширина выемки, внешних откосов и прилегающей полосы, предназначенной для выполнения работ по содержанию дороги, которая принимается не менее 3 м с каждой стороны. Для дорог при высоте насыпи до 2 м крутизну откосов принимают не круче 1:3.

### 2.2. Определение размеров насыпи

Ширина насыпи определяется по расчётной схеме, приведённой на рис. 2.1.

*Рис. 2.1. Расчётная схема для определения средней высоты и ширины насыпи.*

Средняя ширина насыпи поверху:

**В_ср = В + 2(h_д.о · m), м,** (2.3)

где В — нормативная ширина насыпи поверху, принимаемая в зависимости от категории дороги; m — коэффициент заложения откосов насыпи; h_д.о — толщина дорожной одежды.

### 2.3. Определение размеров боковых резервов

Размеры боковых резервов определяются из расчётной схемы (рис. 2.2).

*Рис. 2.2. Расчётная схема для определения размеров боковых резервов.*

Ширина резерва поверху:

**b₁ = W/(2h₀) + (m + n)·h₀/2, м,** (2.4)

Ширина резерва понизу:

**b₂ = W/(2h₀) − (m + n)·h₀/2, м,** (2.5)

где b₁ — ширина резерва поверху; b₂ — ширина резерва понизу; h₀ — средняя глубина резерва; m, n — коэффициенты заложения откосов насыпи и резервов; W — площадь поперечного сечения резерва, м².

Площадь поперечного сечения резерва:

**W = (В_ср + m·h_ср)·h_ср · K_у^отн, м²,** (2.6)

где K_у^отн — коэффициент относительного уплотнения грунта, зависящий от свойств грунта и коэффициента уплотнения грунтов в земляном полотне. Принимается равным 1,1.

### 2.4. Определение дальности перемещения грунта

Среднее расстояние перемещения грунта из боковых резервов в насыпь:

**L_ср = (b₁ + В_ср)/4 + h_ср · m, м.** (2.7)

### 2.5. Определение объёмов работ

По результатам расчётов с учётом категории дороги определяется поперечный профиль земляного полотна. С учётом длины возводимого земляного полотна определяется объём работ.

Объём работ по снятию растительного грунта:

**F_р.с.г = (В_ср + m·h_ср + b₁) · 2L, м²,** (2.8)

где L — длина участка дороги (на 100 п.м).

Обратная надвижка растительного слоя грунта:

**V_р.с.г = (В_ср + m·h_ср + b₁) · 2L · Δz, м³,** (2.9)

где Δz — толщина снимаемого растительного слоя грунта (принято 0,20 м).

Объём грунта, перемещаемого в насыпь:

**V_ср = ((В_ср + В_н)/2) · h_ср · K_у^отн · L, м³,** (2.10)

где В_н — ширина насыпи понизу:

**В_н = В_ср + 2(m·h_ср), м.** (2.11)

Площадь планировки верха насыпи и дна резервов:

**F_н.р = (В_ср + 2b₂) · L, м².** (2.12)
